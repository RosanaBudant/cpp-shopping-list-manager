# Shopping List Manager (C++)

A terminal-based shopping list manager developed in C++.

This project allows users to add, list, edit, and remove shopping list items while storing data persistently in a TXT file.

---

## Features

- Add new items
- List saved items
- Edit existing items
- Remove items
- Persistent file storage
- Interactive terminal menu

---

## Technologies Used

- C++
- STL (`vector`, `string`)
- File handling with `fstream`
- Object-oriented programming

## How to Compile

### Linux / Mac

```bash
g++ main.cpp -o shopping_list
./shopping_list
```

### Windows (MinGW)

```bash
g++ main.cpp -o shopping_list.exe
shopping_list.exe
```
